- style.css      → All styling (REQUIRED — keep in same folder)

## How to upload to GitHub Pages:
1. Go to github.com → Sign up / Log in
2. Click "New Repository"
3. Name it: khairdeen-pk.github.io
4. Set to PUBLIC → Create Repository
5. Click "uploading an existing file"
6. Drag ALL files from this ZIP into the upload box
7. Create a folder called "images" and upload:
   - logo.png (your Khair Deen logo)
   - bag-front.png (your flour bag image)
8. Click "Commit changes"
9. Go to Settings → Pages → Branch: main → Save
10. Your site is live at: https://khairdeen-pk.github.io
